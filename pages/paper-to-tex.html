SECTION 01

为什么 需要转换

学术投稿、毕业论文、会议投稿 — LaTeX 几乎是刚需，但手动转换费时费力。

格式要求不同

多数期刊和会议要求 LaTeX 投稿，但你的初稿可能是 Markdown 或 Word。手动重写格式极其痛苦。

耗时且易出错

手动转换公式、参考文献、图表编号，一篇 20 页的论文可能需要 1-2 天。而且很容易出错。

Markdown / Word

写起来快、方便协作、适合初稿。但缺少专业排版、公式渲染受限、无法满足期刊模板。

LaTeX

学术排版标准、公式渲染完美、参考文献自动管理。但语法复杂、学习曲线陡峭。

! 问题核心：内容已经写好了，只是格式需要转换。这正是 Claude Code 最擅长的事情。

SECTION 03

实操 演示

完整演示如何用 Claude Code 将一篇 Markdown 论文转为 LaTeX。

Step 1 — 准备论文文件

确保你的论文文件在当前项目目录下。支持两种格式：

.md Markdown 格式

适合技术博客、笔记型论文。Claude Code 能直接读取和解析。

.docx Word 格式

需要先安装 pandoc 进行预转换，或让 Claude Code 通过 Skill 处理。

Step 2 — 给 Claude Code 下达指令

在终端中启动 Claude Code，然后描述你的需求：

Claude Code Terminal 
$ claude

Claude Code v1.0

> 把 paper.md 转成 LaTeX 格式，要求：
  1. 使用 IEEE 双栏模板（IEEEtran）
  2. 保留所有数学公式，转成 LaTeX 语法
  3. 图片引用保持不变，用 \includegraphics
  4. 参考文献用 \cite{} 格式
  5. 生成完整的 .tex 文件

Step 3 — Claude Code 自动执行

Claude Code 会自动完成以下工作：

读取源文件
分析 paper.md 的结构：标题、段落、公式、图表、引用。
生成 LaTeX 骨架
创建 paper.tex，添加文档类、宏包、页面布局等模板配置。
转换内容
逐节转换：标题 → \section{}，列表 → itemize，公式保持不变。
处理引用
识别文中引用，生成 \cite{key} 格式，创建 references.bib。

SECTION 04

模板 配置

常用 LaTeX 模板的 Claude Code 指令，直接复制使用。

IEEE 会议论文

Claude Code Prompt 
将 paper.md 转为 LaTeX，使用 IEEEtran 会议模板。要求：
1. 双栏排版，A4 纸张
2. 包含 \usepackage{amsmath,amssymb,graphicx,hyperref}
3. 标题居中，作者信息用 \author{} 格式
4. 摘要用 abstract 环境
5. 图表自动编号，用 \ref{} 引用

Elsevier 期刊论文

Claude Code Prompt 
将 paper.md 转为 LaTeX，使用 Elsevier 模板。要求：
1. 文档类 \documentclass[preprint]{elsarticle}
2. 包含 elsarticle.cls 模板宏包
3. 使用 \begin{frontmatter} 包裹标题、作者、摘要
4. 关键词用 \keyword{} 标记
5. 参考文献用 elsarticle-num.bst 样式

中文毕业论文

Claude Code Prompt 
将 paper.md 转为中文毕业论文 LaTeX 格式。要求：
1. 使用 ctexart 文档类，UTF-8 编码
2. 设置宋体/黑体/楷体字体映射
3. 章节格式：第一章、1.1、1.1.1
4. 页眉页脚符合学校要求
5. 摘要分中文/英文两部分
6. 目录用 \tableofcontents
i 如果你的学校/期刊有特定的 .cls 或 .sty 模板文件，把它放到项目目录下，然后告诉 Claude Code：使用 mytemplate.cls 模板

SECTION 05

进阶 技巧

让转换结果更好的实用技巧。

提供模板文件

将目标期刊的 .cls 或 .sty 文件放入项目目录，Claude Code 能精确匹配模板要求。

分步转换

对于长论文，可以先转骨架（标题+章节），再转公式，最后处理参考文献。逐步完善。

编译验证

转换后让 Claude Code 执行 pdflatex 编译，自动修复语法错误。

增量修改

论文修改后，只需让 Claude Code「更新 tex 文件」，它会对比差异后局部更新。

实用 Prompt 模板

> 分步转换：先帮我把 paper.md 的目录结构转成 LaTeX 骨架，只保留章节标题，不要转内容
> 编译修复：编译 paper.tex，如果有错误帮我修复，直到编译通过
> 公式检查：检查 paper.tex 中所有数学公式，确保语法正确
> 参考文献：从 paper.md 中提取所有引用，在 paper.bib 中生成 BibTeX 条目
✓ 掌握这些技巧后，一篇 20 页的论文转换只需几分钟。去 配置指南 确保你的 Claude Code 已就绪！
