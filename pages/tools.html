CC-Connect

AI Coding Agent ↔ 消息平台桥梁

CC-Connect 是一个开源桥接工具，将本地运行的 AI 编程助手（Claude Code、Cursor、Gemini CLI、Codex 等）连接到你常用的即时通讯平台。无需公网 IP，直接在飞书、钉钉、Slack、Telegram 中与 AI 搭档实时对话，远程下达编程指令。

🤖 Claude Code 本地终端
↔
⚡ CC-Connect 桥接层
↔
💬 消息平台 飞书 / 钉钉 / Slack ...

支持的 AI Agent

Claude Code Cursor Gemini CLI Codex

支持的消息平台

飞书 / Lark 钉钉 Slack Telegram Discord LINE 企业微信

无需公网 IP

大多数平台支持内网穿透，本地即可运行

🔌

CC-Switch

Claude Code 模型配置管理器

CC-Switch 是一个桌面端可视化工具，让你无需手动编辑 JSON 配置文件，通过图形界面即可管理 Claude Code 的模型设置。一键切换 DeepSeek、MiMo 等不同平台的模型，支持多配置方案快速切换。

核心能力

🖥️

图形化配置

告别手写 JSON，可视化编辑 Base URL、模型等参数

⚡

一键切换模型

在 DeepSeek、MiMo 等多个配置方案间瞬间切换

🔍

连通性测试

内置测试按钮，配置完成后立即验证 API 是否可用

📋

多配置管理

同时保存多套配置，工作/学习/个人场景灵活切换

支持的模型平台

DeepSeek Xiaomi MiMo Anthropic (官方) 更多兼容平台
