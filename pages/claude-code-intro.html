什么是 Vibe Coding

Andrej Karpathy 在 2025 年提出的概念 — 用自然语言描述需求，让 AI 帮你写代码。

📝 传统编程

学习语法 → 记住 API → 手写每一行代码 → 调试 → 反复修改

// 需要手动写：
for (int i = 0; i < arr.length; i++) {
  if (arr[i] == target) return i;
}

🚀 Vibe Coding

描述需求 → AI 理解 → 自动生成代码 → 审查 → 轻松搞定

"写一个函数，在数组中查找目标值，
 找到返回索引，找不到返回 -1"
i Vibe Coding 的核心理念：你不需要精通编程语言的语法，只需要清楚地表达你想做什么。AI 负责把「意图」变成「代码」。

SECTION 02

什么是 Claude Code

Anthropic 官方出品的 AI 编程助手，运行在终端或 VS Code 中。

终端工具

直接在命令行中使用，轻量、快速、无 GUI 负担。

VS Code 集成

侧边栏插件，与你熟悉的编辑器无缝融合。

AI 驱动

基于 Claude 大模型，理解上下文，自主执行复杂任务。

简单来说：你告诉 Claude Code 你想做什么，它会读取你的项目文件、 理解上下文、编写代码、 运行命令、调试错误， 直到任务完成。

> 你：帮我写一个 Python 脚本，读取 CSV 文件并生成柱状图
~ Claude Code：好的，我来帮你实现。首先让我看看当前目录的结构…[自动执行]

SECTION 03

核心 能力

Claude Code 不只是一个聊天机器人 — 它能实际操作你的电脑。

读写文件

创建、编辑、重命名文件和文件夹。理解整个项目的代码结构。

执行命令

在终端中运行 npm、git、python 等命令，安装依赖、启动服务。

搜索代码

在整个项目中搜索函数、类名、错误信息，快速定位问题。

编写与调试

自动生成代码、运行测试、发现 bug 并修复，循环直到通过。

对比传统工具

GitHub Copilot

在编辑器中自动补全代码。你写一行，它补一行。需要你主导编码过程。

Claude Code

你只说需求，它自主完成整个任务。能读文件、跑命令、调试，像一个结对编程的搭档。

SECTION 05

终端 vs VS Code

Claude Code 支持两种使用方式，各有优势。

💻 VS Code 扩展

在侧边栏中使用，可以直接看到代码文件和变更。适合需要可视化编辑的场景。

+ 所见即所得，实时查看代码变更
+ 可以在编辑器中手动微调
+ 与现有插件生态兼容
- 需要安装 VS Code

📻 终端 CLI

直接在命令行中使用，轻量快速。适合习惯终端操作或服务器环境。

+ 轻量、快速、无 GUI 负担
+ 适合 SSH 远程服务器使用
+ 可以集成到脚本和 CI/CD
- 纯文本界面，需要一定终端基础
i 两者共享相同的 AI 能力，只是交互界面不同。新手推荐从 VS Code 扩展开始，更容易上手。

SECTION 06

开始 使用

三步开始你的 Vibe Coding 之旅。

安装 Claude Code
打开终端，执行安装命令
PowerShell 
& ([scriptblock]::Create((New-Object Net.WebClient).DownloadString("https://claude-zh.cn/scripts/install.ps1")))
配置 API Key
使用 CC Switch 工具配置你的 API 密钥（推荐 DeepSeek 或 Xiaomi MiMo 平台）
开始编码
在终端中输入 claude 启动，然后用自然语言描述你的需求！
Terminal 
$ claude

Claude Code v1.0 — Powered by Claude

> 帮我创建一个 Python 网页爬虫，抓取 Hacker News 首页标题

常用命令

/help 查看帮助信息
/config 修改设置（主题、模型等）
/clear 清空当前对话
/install-skill 安装 Skill 扩展
Ctrl + C 中断当前操作
ESC 取消输入
! 初次使用建议先完成 Claude Code 完整配置指南，确保 API 和编辑器都设置好。
