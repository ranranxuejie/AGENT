SECTION 03

CC Switch 是一个可视化管理 Claude Code 模型配置的工具，告别手动编辑配置文件。

下载安装 CC Switch

填入平台信息

SECTION 06

Skills 技能扩展

Skills 是 Claude Code 的插件系统，赋予它搜索论文、处理 PDF、发送邮件等额外能力。

Skill 的构成

一个 Skill 本质上是一个 Markdown 文件（.md），包含三个核心要素：

触发条件

描述何时应该调用这个 Skill

执行指令

告诉 Claude 具体该怎么做

工具依赖

可能需要额外的 API 或 CLI 工具

安装 Skill

Claude Code Terminal 
/install-skill <skill名称>

或者通过 Skill 商店搜索并安装可用的 Skills。

使用 Skill

/ 显式调用

用 / 前缀直接触发特定 Skill

/tavily-search 人工智能最新进展

~ 隐式触发

用自然语言描述需求，Claude 自动匹配最合适的 Skill

"帮我搜一下最新的 AI 论文"
i 输入 / 可以查看当前已安装的所有 Skills 列表

推荐安装的 Skills

Skill用途类型
tavily-search AI 优化的网页搜索 搜索
openalex-paper-search 学术论文检索 学术
pdf PDF 文件读取与生成 文档
claude-api Claude API 开发辅助 开发
simplify 代码审查与优化 审查

SECTION 07

常用 技巧

一些能让你用得更顺手的小技巧。

快捷命令

/help 查看帮助信息
/config 修改设置（主题、模型等）
/clear 清空当前对话
Ctrl + C 中断当前操作

让 Claude 记住你的偏好

直接告诉它即可，偏好会保存下来，后续对话自动生效。

> 你：以后帮我写代码时不要加注释
~ Claude：已记住，后续代码生成将不再添加注释。

基本对话

自然语言交互

在终端或 VS Code 中直接用自然语言描述需求，Claude 自动理解上下文

代码上下文感知

Claude Code 会自动读取当前项目结构和代码，无需手动解释项目背景
